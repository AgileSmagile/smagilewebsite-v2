# smagile.co

Company website for smagile, built with Astro 5 and Tailwind CSS 4.

## Local development

```sh
npm install
npm run dev        # http://localhost:4321
```

## Build and preview

```sh
npm run build
npm run preview
```

## Deployment

Push to `main` to deploy to production. Push to `develop` for staging. GitHub Actions handles the rest (build, package, deploy to Pi via Cloudflare tunnel).
